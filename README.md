# market-Preview

# 技术栈
框架: Next.js (App Router 模式)

语言: TypeScript

样式: Tailwind CSS + CSS 变量主题系统

UI组件库: shadcn/ui (基于 Radix UI)

图标: Lucide React

部署: Vercel Analytics 集成

# 项目类型
这是一个 校园二手交易与悬赏求助平台 的移动端Web应用，主要功能包括：

商品/帖子浏览（瀑布流布局）

发布功能

消息系统

个人中心

安装依赖
npm install

启动开发服务器
npm run dev

访问项目
打开浏览器，访问 http://localhost:3000
